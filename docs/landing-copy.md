# Продающий текст для главной страницы

## Заголовок

Pushka — пушка для агентов: слать и планировать напоминания

## Тело

Агент видит важное — дедлайн, результат долгой задачи, повод написать клиенту — но сказать тебе
об этом может только внутри чата, который ты, скорее всего, уже закрыл. **Pushka** даёт агенту
один инструмент на два сценария: выстрелить пушем прямо сейчас или зарядить его на потом.

Срочно напомнить — агент шлёт push немедленно.
Напомнить через час, завтра или через неделю — агент планирует push на нужное время, а Pushka
сама выстрелит в момент.

Один и тот же вызов, разница — в одном поле. **Уже сегодня:** curl-friendly REST API. **Скоро:**
Skill и MCP-сервер — агенту вообще не придётся знать, что такое API-запрос.

## Как это работает

1. Разрешаешь уведомления в браузере — один раз.
2. Даёшь агенту свою push-подписку.
3. Агент решает — сейчас или потом — и напоминание долетает.
